# 🧠 AI-Based Early Disease Prediction System

## 📌 Overview
The **AI-Based Early Disease Prediction System** is a machine learning-based application that predicts possible diseases based on user-input symptoms.

Users can enter multiple symptoms, and the system provides a list of potential diseases along with their probability percentages. This enables **early awareness and timely medical attention**, helping reduce the risk of severe health conditions.

---

## 🎯 Objectives
- Predict diseases at an early stage using symptoms
- Provide probability-based results for better understanding
- Assist users in making timely medical decisions
- Build an easy-to-use and accessible system

---

## ❗ Problem Statement
Many people tend to ignore early symptoms, which can lead to serious health issues.

There is a need for a system that can quickly analyze symptoms and suggest possible diseases for early detection.

---

## 💡 Solution
Our system uses a **Machine Learning model** that:
- Accepts multiple symptoms as input
- Processes and analyzes the data
- Predicts possible diseases
- Displays top predicted diseases with probability percentages

---

## ⚙️ How It Works
1. User enters multiple symptoms (up to 5)
2. Symptoms are processed and encoded into model input
3. The trained ML model analyzes the data
4. The system predicts possible diseases
5. Results are displayed with probability percentages

---

---

## 🧪 Features
- Input multiple symptoms
- Predict top possible diseases
- Display probability (%) for each disease
- Fast and user-friendly interface

## 📊 Example Output

**Input:**  
Fever, Cough, Headache  

**Output:**  
- Flu → 70%  
- COVID-19 → 20%  
- Common Cold → 10%
